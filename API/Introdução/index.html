O que é Scrum?

Scrum é um framework ágil utilizado para desenvolvimento de
produtos complexos. Ele possui uma abordagem incremental, ou
seja, seus projetos são desenvolvidos com várias atividades em
paralelo, que são integradas quando estão finalizadas. Além disso,
o Scrum é iterativo, o que significa que ele é desenvolvido com
um ciclo de vida que contém várias repetições. Dessa forma, é
permitido que o produto vá e volte conforme estiver sendo
desenvolvido.

De acordo com o Guia do Scrum, essa metodologia é leve, simples
de entender, mas extremamente difícil de dominar.
